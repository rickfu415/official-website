A1
技术金字塔

0 → 5：从学科基础到极速制造

0｜学科基础数学 / 物理 / 化学 / 计算机 / 验证
1｜求解器传热 / 流场 / 烧蚀 / 热应力断裂 / 耦合加速
2｜基础架构烧蚀耦合 / TPS 材料 / 高超流场 / 全尺度模拟

核心主线

学科基础 → 核心求解器 → 多物理基础架构 → 行业平台 → 下一代平台 → 设计制造一体化。

能力目标

以高可信多物理仿真为核心，支撑材料研发、结构设计、方案优化与工程交付。

战略含义

技术壁垒不是单个模型，而是跨越求解器、平台和制造闭环的系统性能力。

A2
基础能力层

0 ~ 2：从基础学科到多物理底座

0｜学科基础

数值分析、有限体积 / 有限元、Gas Dynamics、传热、弹性力学、材料学、多孔材料，以及 C++ / Python / MPI / GPU 等计算实现能力。

1｜核心求解器

覆盖传热、超声速流场、烧蚀、热应力与断裂，以及流固 / 热力耦合和异构加速等关键求解能力。

2｜基础架构

把烧蚀多物理耦合、TPS 材料性能、高超声速流场分析和全尺度模拟组织成可复用的工程底座。

底层价值

这一层决定了模型可信度、验证能力和可扩展性，是后续平台化与产品化的真实护城河。

A3
平台跃迁层

3 ~ 5：从计算平台走向下一代 CAE 与极速制造

3｜计算平台

N 平台与物理 AI 求解器沉淀高质量仿真数据，提升高保真分析效率，并把多类物理过程纳入统一框架。

4｜下一代平台

新一代 CAE 平台追求秒级多物理实时反馈、隐式几何表达和自动化设计—仿真—优化闭环。

5｜极速制造

把仿真从验证环节前移为设计引擎，再后接制造执行，最终形成设计及仿真、仿真及制作的一体化流程。

这一层体现的不是单一软件升级，而是研发模式升级：仿真成为设计反馈核心，平台成为效率放大器，制造成为闭环终点。
